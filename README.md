## Space Mission Analysis: Earth-to-Asteroid Mission
This repository contains a suite of MATLAB scripts and functions developed for the analysis and simulation of a complete space mission. The project focuses on a mission trajectory from Earth to a Sun-orbiting Asteroid, covering all phases from departure to arrival.

### Mission Architecture
The project is structured into three primary orbital scenarios and the impulsive maneuvers required to transition between them:

1. Earth Parking Orbit (Departure Phase)
Analysis of the initial circular or elliptical parking orbit around the Earth.

2. Heliocentric Transfer (Interplanetary Phase)
Modeling the spacecraft's orbit around the Sun.


3. Asteroid Orbit (Arrival Phase)
Insertion maneuvers into the target orbit around the asteroid.
